== Description ==
Purple Play Lite is a simple, minimal, responsive, easy to use, one click install, beautiful and Elegent WordPress Theme with Easy Custom Admin Options Created by SketchThemes. Using these theme options any one can easily customize this theme according to their need. You can use your own Logo, Site Title, custom favicon, add social links, rss feeds, you can use own copyright text etc. And all this information can be entered using Theme Options Panel. You have to just set the content from the Themes Options Panel and it'll be up ready to use.


== Frequently Asked Questions ==

= How to set Theme Front Page? =
Go to "Settings >> Reading >> Front page displays" select "A static page (select below)" and there set following
"Front Page" - deafult front page for your site
and "Blog Page" - page on which the blog posts will be displayed

= How to customize the Theme? =
Go to "Appearance -> Customize", you can change the settings and preview them... to save the settings and apply changes click on "Save and Publish".

= How to add Featured Boxes on Front Page? =
Go to "Appearance >> Widgets" and add "Featured Box - PurplePlay Lite" widget in "Home Featured Sidebar".

= How to create a full width page? =
1. Go to Pages.
2. Click Add New or Edit an existing page.
3. Select "Full Width Template" from "Page Attributes >> Template".

= What is preferred size for Logo Image? =
The preferred size for Logo Image is 270x80 pixels.

= How I can use the theme in another language? =
The PurplePlay Lite theme is translation ready. The translations file are in language folder. For more details you can read the following blog https://sketchthemes.com/blog/effective-steps-to-translate-your-wordpress-theme-using-qtranslate/
